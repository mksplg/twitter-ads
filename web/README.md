= Twitter Ads

== Installation

1. Install node.js
2. Install npm

3. Install grunt
    `npm install -g grunt-cli`
3. Install local grunt
    `npm install grunt`
4. Build project
    `npm install`

=== Run for production
	Execute grunt production and then run index
    `npm start`

=== Run for development
	Will grunt test, grunt development and grunt watch
    `grunt`
